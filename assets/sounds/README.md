# 音效素材來源說明

本資料夾存放《Escape the Equation：教室危機》使用的遊戲音效檔案。

這些音效用於提升遊戲互動回饋，包含答對、答錯、出口門解鎖、成功逃脫與遊戲失敗等情境。

## 檔案說明

- `correct.mp3`  
  玩家答對謎題時播放，用於提供正向回饋。

- `wrong.mp3`  
  玩家輸入錯誤答案時播放，用於提示答題失敗。

- `unlock.mp3`  
  玩家收集完五個密碼碎片、出口門解鎖時播放。

- `success.mp3`  
  玩家成功輸入最終密碼並逃離教室時播放。

- `gameover.mp3`  
  玩家因時間歸零或生命值歸零而失敗時播放。
